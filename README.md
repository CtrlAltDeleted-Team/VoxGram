# VoxGram
VoxGram — анонимный мессенджер не собирает никакие данные, кроме никнейма, пароля и IP-адреса. Премиум-подписка в сервисе никак не влияет на вашу анонимность и добавляет только визуальные функции, снимает ограничения на размер файлов и количество папок для чатов.
# Доступные платформы
Android Windows и Web Версия доступная на сайте voxgram.ru
Скачать приложение можно на voxgram.ru/info.php
# Политика закрытого исходного кода
Исходный код VoxGram является полностью закрытым. Это осознанное решение для минимизации технических рисков. Отсутствие кода в открытом доступе предотвращает:
Реверс-инжиниринг и поиск векторов атак злоумышленниками.
Эксплуатацию потенциальных «нулевых» уязвимостей до их исправления.
Создание фейковых модификаций (скам-клиентов), крадущих данные пользователей.
# Безопасность и Анонимность
VoxGram не требует номер телефона или email. Идентификация происходит только по выбранному никнейму. Мы соблюдаем законодательство, логируя только IP-адреса, что при использовании VPN гарантирует пользователю полную конфиденциальность.
